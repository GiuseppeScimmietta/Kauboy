Pre-Lexing
--------------------------

Purpose of Pre-Lexing is to find kauboy specific keywords (statements) within the source code and generate list of lexing tokens

(STATEMENT, value, lineNo, charNo)

e.g.
(EXPECT_STM, "expect", 10, 2)

PreLexing
--------------------------

Purpose of prelexinf is to find kauboy specific keywords (statements) within the source code an generate list of lexing tokens

(STATEMENT, value, lineNo, charNo)

e.g.
(EXPECT_STM, "expect", 10, 2)
